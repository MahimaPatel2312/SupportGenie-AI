# 🤖 SupportGenie AI

An AI-powered Customer Support Assistant built using Flask, SQLite, HTML, CSS and JavaScript.

## Features

- User Authentication
- FAQ-based Intent Detection
- Customer Support Chatbot
- SQLite Database
- Conversation Management
- Dashboard
- Modern Chat UI
- Typing Animation

## Tech Stack

- Python
- Flask
- SQLite
- HTML
- CSS
- JavaScript
